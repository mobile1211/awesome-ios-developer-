## JSON
JSON is a language-independent data format
> Which is relative with **KEY - VALUE** pair
```json
{
    "main": [
        {
            "title": "example1",
            "body": "body example1"
        },
        {
            "title": "example2",
            "body: "body example2"
        },
    ]
}
```
